# NPM

- Node Package Manager


## 1. 모듈설치
~~~
npm install <모듈 이름>
~~~

### 1.1글로벌설치
시스템 디렉토리에 설치한다.
~~~
npm install <모듈 이름> -g
~~~

### 1.2로컬설치
해당 디렉토리에 node_modules폴더에 설치하는걸 의미한다.


## 2. 모듈제거
~~~
npm uninstall express
~~~

## 3. 모듈 업데이트
~~~
npm update express
~~~

## 4. 모듈 검색
~~~
npm search express
~~~


### package.json
프로젝트가 의존하는 모듈과 모듈버전정보를 담고 있다.
