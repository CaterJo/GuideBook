# webpack Development Server

## Intro
node express로 만들어진 webserver다.  
파일의 변경을 감지해서 리 컴파일한다. 추가적인 미들웨어를 설정해서 넣을 수도 있다.


## 개발서버 설치
```
npm install –dev webpack-dev-server
```
