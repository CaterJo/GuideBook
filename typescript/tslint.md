## intro

강제 적용을 위해서 Pretter를 설치해야한다.

1. vscode 프리티어 확장 설치
2. npm i prettier설치
3. package.json

```json
"prettier": { "printWidth": 80, "useTabs": false, "tabWidth": 2, "bracketSpacing": true, "semi": true, "singleQuote": false } }

```

4. setting.json

```json
(editor.formatOnSave = true)
```
