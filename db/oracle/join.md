## Join
    서로 다른 테이블간에 설정된 관계가 결합하여 1개 이상의 테이블에서 데이터를 조회하기 위해 사용 됩니다. 
    이 때 테이블간의 상호 연결을 조인이라고 하는데요. 
    각각의 테이블에 분리된 연관성 있는 데이터를 연결하거나 조합해야 하는데 이러한 일련의 작업들을 조인이라고 합니다.

- 내부 조인 ( Inner Join )

- 외부 조인 ( Outer Join )

- 크로스 조인 ( Cross Join )

- 셀프 조인 ( Self - Join )

- 안티 조인 ( Anti Join )

- 세미 조인 ( Semi Join )

- ANSI 조인
