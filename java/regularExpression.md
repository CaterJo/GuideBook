# Regular expression


## Regular expression flags
- g: 전역검색(Global)
- i: 대소문자 구분없이 검색
- m: 다중행검색
- u: 유니코드; 패턴을 유니코드패턴의 나열로 취급
- y: 'sticky'검색을 수행한다.

## 특수문자. 

- ^ : it assert Start position or 역치
- $ : it assert End position
- \* : it is meaned repeat zero to unlimited times
