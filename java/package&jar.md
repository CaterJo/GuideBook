		*path는 실행프로그램을 윈도우 아무곳에서나.

##JAR 기본명령어		
- c (압축) :새 아카이브를 생성합니다.
- v	표준 출력에 상세정보 출력을 생성합니다.
- f	아카이브 파일 이름을 지정합니다. (저장할 파일명을 지정할 수 있다)

jar cvf 파일명.jar 클래스명1.class 클래스명2.class
(파일명으로 압축하겠다. 클래스1과 클래스2를)
