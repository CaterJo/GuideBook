# VSCODE

## 1. 단축키

- 텍스트 찾기       : ctrl + F
- 텍스트 바꾸기     : Ctrl + R
- 파일찾기          : Ctrl + P
- 사이드바 토글     : Ctrl +B
- 명령어 패널 열기  : Ctrl + Shift + p
- 멀티커서          : Ctrl + Alt + UP || Dwon
- 코드 정렬         : Shift Alt F
- 멀티라인 주석 : Shift + Alt + A
- 수정했던 라인으로 이동 : Alt + 좌우방향키
- 설정화면열기 : Ctrl + ,
